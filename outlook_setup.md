# Outlook.com Free Email Setup

## Configuration for .env:
```env
SMTP_HOST=smtp-mail.outlook.com
SMTP_PORT=587
SMTP_USER=your-email@outlook.com
SMTP_PASSWORD=your-outlook-password
FROM_EMAIL=your-email@outlook.com
```

## Setup Steps:
1. Create free Outlook account at https://outlook.com
2. Go to account settings
3. Enable "Less secure app access" or use app password
4. Use your Outlook email and password

## Alternative - Hotmail:
Same settings work for @hotmail.com addresses

## Limits:
- 300 emails per day
- Free forever
